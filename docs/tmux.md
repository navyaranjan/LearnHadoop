# Tmux Docs

TMUX is a terminal multiplexer. It helps in creating multiple splits inside the terminal.

## Launching tmux

After opening `Gnome-Terminal` launch tmux using `tmux` command.

```
$ tmux
```

## Horizontal Split

```
Ctrl+b % 
```

## Vertical Split

```
Ctrl+b "
```

## Switch between splits

```
Ctrl+b UP/DOWN/LEFT/RIGHT
```
